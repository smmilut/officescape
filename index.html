goal :

Work spawns from xerox machines. Hit work with your attack to perform the work.

input :

keyboard or gamepad with following bindings :

  • move : arrow keys, or A/W/S/D, or Z/Q/S/D, or gamepad directions
  • attack : h, or gamepad button X
  • (not implemented) start : Enter, or gamepad button Start

debug :

The current seed of the RNG is displayed in the browser developper tools console.

To load the game using a specific RNG seed, add the URL parameter ?seed=123 where 123 should be replaced by whatever is your seed.
